# Management QuickPlan

Management QuickPlan is a lightweight project management system developed for SWE 4663. 
The application is designed to help users manage project information, requirements, risks, effort logs, 
and summary reporting through a simple web-based interface.

## Features

- Create and manage project information
- Record functional and non-functional requirements
- Track project risks and mitigation strategies
- Log effort by development phase
- View dashboard summaries and reports

## Tech Stack

- Python
- Flask
- Pandas
- CSV file storage
- HTML/CSS

## Project Structure

- `app/` - application logic, routes, templates, and static files
- `data/` - CSV files used for storage
- `docs/` - project documents, mockups, and diagrams
- `tests/` - test files

## Running the Project

1. Clone the repository
2. Create and activate a virtual environment
3. Install dependencies from `requirements.txt`
4. Run the Flask application
